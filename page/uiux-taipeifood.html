Story

Deciding restaurant is always the bigest problem that we have everyday.Specialy we wanted to have meal with friends or date.

We dont want them to feel we are hesitant person.So this website provide a radom restaurant selecter that can choose what should you eat for you tonight.

Project Goals

  • Developing a website connect with backend sofeware Firebase.
  • Practicing Javascript apply to control website.
  • RWD website practicing.

JS & Backend Sofeware

This project is one of the homework,when I was selflearning from website.

I only knew how to use frontend coding to make website in that time, but after learned how to use firebase.I don't need to code every pages for each data.Let the sofeware calcilation for me.

Although after searching more on Internet,I know firebase is more suit for prototype.But it already give me a change, which I can finish a project by myself. It was a good exeperience for a industrial designer.

It's just a simple web,include random restaurant selecter and information, a page you can review all the restaurant in the database,and a page let you update data to firebase.

The home pages,user can easily get a random restaurant and the information by click a button.

These two pages are for user to edit database.By seting category of restaurant or dating style helping database to classify which type of store you put.
