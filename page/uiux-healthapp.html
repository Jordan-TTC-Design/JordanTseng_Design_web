Story

This project was the frist interface design I did when I was a college student.
Coorperating with a company which wanted to provide a service connect insurance agency,doctor and patient. By using this application and a detector tied on body,patient's real time data and status will notify agency and doctor immediately.

Project Goals

  • Develop application's interface for patient and insurance agency.
  • Design an interface that engineer can complete.
  • Design logo and website interface.

Before we started to do it with computer,we discuss with team member.We drew a lot of manuscript to compare the frame and the using process.

Also we set up all the function's location,combine some of them into one page might be better for elder.

Design Process

Actually we made many style of interface like upper picture,the right one was what client want.

Base on chromatology, cyan probably won't be the best color for elder. But Client think is more suit for the application related with medical.

This application include basic sport app function and it can detect your heartbeat , breathe , body temperture and even your body posture.This appliciation connect with insurance angency directly,they will call 911 if you had any accident.
